3인칭 액션 rpg 게임을 제작을 위한 저장소 입니다.
c++ 위주로 작업을 진행해서 전체적인 언리얼엔진 사용법과 구현능력을 기르기 위한 프로젝트 입니다.

넥슨의 빈딕투스-디파잉페이트의 프리알파 테스트를 체험을 통해 아이디어를 얻었고 
해당 게임을 모작하는 프로젝트 입니다.
그래서 프로젝트의 이름은 빈딕투스의 V 를 따서 V1 으로 이름을 지었습니다.
